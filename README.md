# FindWeather App

### Technologies

- React Native
- Typescript
- Axios
- Expo
- Styled-components
- React-navigation

### How To Use

```bash
# Clone this repository
$ git clone https://github.com/HelenCrisM/react-native-find-weather/

# Go into the repository
$ cd react-native-find-weather

# Install npm
$ npm i

# Run the app 
$ npm start
```
